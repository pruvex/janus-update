# Skill: /debug-log
Debug-Log Analyse für schnelle Diagnose via D11 Debug Compression Engine.

**Ziel:** Komprimierte Diagnose der letzten 10 Minuten Logs für AI Studio.

## Nutzung
Tippe in Windsurf: `/debug-log`

Der Workflow ruft automatisch den Debug-Endpunkt auf und zeigt das Ergebnis im Chat an.

## Ausführung

```bash
curl.exe -s http://localhost:8000/api/system/debug-summary
```

## Rückgabe
Die Engine analysiert die letzten 10 Minuten Logs und gibt eine Markdown-Zusammenfassung zurück mit:
- Heuristic Findings (Hard Errors, Model Drift, Latency Spikes)
- Confidence Score
- Quick Summary für schnellen Überblick
