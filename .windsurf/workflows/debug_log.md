# Port 8001: Janus Backend Production Port

# Skill: /debug-log
Debug-Log Analyse für schnelle Diagnose via D11 Debug Compression Engine.

**Ziel:** Komprimierte Diagnose der letzten 10 Minuten Logs für AI Studio.

## Nutzung
Tippe in Windsurf: `/debug-log`

Der Workflow ruft automatisch den Debug-Endpunkt auf und zeigt das Ergebnis im Chat an.

## Ausführung

```bash
curl.exe -s -X POST http://localhost:8001/api/skills/debug-log -H "Content-Type: application/json" -d '{\"mode\":\"fast\"}'
```

Optionale Parameter:
- `trace_id`: Filter logs by specific trace ID
- `mode`: "fast" (100 logs, default) or "full" (500 logs)

## Rückgabe
Die Engine analysiert die letzten 10 Minuten Logs und gibt ein JSON-Objekt zurück mit:
- **d11_debug_report**: Strukturierter Debug-Report
  - **generated_at**: Timestamp der Analyse
  - **logs_analyzed**: Anzahl der analysierten Logs
  - **confidence_score**: Confidence Score (0.0 - 1.0)
  - **hard_errors**: Anzahl der Hard Errors
  - **model_drift**: Boolean ob Model Drift erkannt wurde
  - **latency_spikes**: Anzahl der Latency Spikes
  - **suggest_d13**: Boolean ob D13 Optimization Report empfohlen wird (True bei hard_errors > 2 oder latency_spikes > 1 oder model_drift == True)
  - **markdown_report**: Formatierte Markdown-Zusammenfassung mit:
    - SUMMARY: High-level Überblick
    - ROOT CAUSE: Technische Ursachenanalyse
    - FINDINGS: Detaillierte Heuristik-Ergebnisse (Hard Errors, Model Drift, Latency Spikes)
    - CONFIDENCE: Confidence Score mit Interpretation
    - RECOMMENDED ACTION: Konkrete Handlungsempfehlungen

## Cascade Integration

**Anleitung für Cascade:**

1. **JSON empfangen:** Der Endpoint gibt ein JSON-Objekt zurück, nicht nur reinen Text.
2. **Summary anzeigen:** Zeige dem User den `markdown_report` Inhalt aus `d11_debug_report.markdown_report`.
3. **suggest_d13 Prüfen:** Wenn `d11_debug_report.suggest_d13` auf `true` steht, füge folgenden Hinweis hinzu:
   ```
   ⚠️ Systemic issues suspected. Run /optimization-report for deep dive.
   ```

## Ziel
Kopiere die Diagnose direkt in AI Studio für schnelles Debugging. Bei suggest_d13 = True empfiehlt sich ein D13 Deep Dive.
