# Skill-Spezifikation: system.price_comparison (V1.1)

## 1. Funktionale Vision
Hochpräziser globaler Finanz-Assistent. Findet den günstigsten Gesamtpreis (Total Cost) durch Vergleich von Elite-Portalen (z.B. Idealo/Geizhals).

## 2. Kern-Kompetenzen
- **Das Preis-Duell:** Vergleicht aktiv Quellen, wählt den Gewinner inklusive Versandkosten.
- **Refurbished-Boost:** Markiert Ersparnisse >= 20% als `[SPAR-TIPP]`.
- **Global Awareness:** Wählt automatisch passende regionale Portale und Währungen.

## 3. Nahtlose Integration
- **Seamless UX:** Schlägt bei Preis-Unklarheiten keine anderen Tools vor, sondern präzisiert das Ergebnis intern.

## 4. Benchmark & Tiering
- **OpenAI:** `gpt-5.4-mini` (Balanced) – 100% Success-Rate, Ø 0.98s Latenz.
- **Gemini:** `gemini-3-flash-preview` (Speed) – 100% Success-Rate, Ø 1.91s Latenz.
